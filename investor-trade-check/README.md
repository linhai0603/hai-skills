# investor-trade-check

Investment Trade Check helps long-term investors slow down before buying, selling, adding, trimming, or waiting.

It turns a trading urge into a written decision check across thesis, evidence, emotion, circle of competence, valuation, margin of safety, position size, funding source, and exit rules.

This skill is for educational decision support only. It does not provide stock picks, price forecasts, buy/sell/hold recommendations, or personalized financial advice.

## Files

- `SKILL.md`: Codex skill instructions and trigger rules.
- `references/`: pre-trade decision framework.
- `templates/`: reusable output structure.
- `examples/`: fictional usage example.
- `agents/`: local skill metadata.
