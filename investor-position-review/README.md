# investor-position-review

Investment Position Review helps long-term investors review whether an existing holding's original thesis still holds after price moves, news, earnings, valuation changes, emotional pressure, or portfolio drift.

It separates market noise from thesis evidence and classifies the position review as intact, watch closely, weakened, broken, or mainly a position-risk issue.

This skill is for educational decision support only. It does not provide stock picks, price forecasts, buy/sell/hold recommendations, or personalized financial advice.

## Files

- `SKILL.md`: Codex skill instructions and trigger rules.
- `references/`: position-review framework.
- `templates/`: reusable output structure.
- `examples/`: fictional usage example.
- `agents/`: local skill metadata.
