# knowledge-system-curator

Knowledge System Curator helps update a living knowledge base by merging new notes, excerpts, documents, or research into the existing structure.

It focuses on coverage, deduplication, source boundaries, outline quality, and preserving important original wording instead of simply appending or summarizing new material.

## Files

- `SKILL.md`: Codex skill instructions and trigger rules.
- `docs/`: skill-specific packaging and promotion notes.
- `examples/`: source-to-output demo workflow.
- `agents/`: local skill metadata.
